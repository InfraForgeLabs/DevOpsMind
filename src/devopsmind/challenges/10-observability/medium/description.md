You are given a file metrics.txt containing lines formatted like:

latency_ms=120
latency_ms=85
latency_ms=200
...

Task:
- Create metrics.py
- Script must:
    - Read metrics.txt
    - Extract all latency_ms values
    - Compute the arithmetic mean
    - Print ONLY the number (e.g., 135.0), no text or extra formatting

This challenge teaches:
- Basic log/metric parsing
- Simple statistical aggregation
- CLI-style Python output discipline
